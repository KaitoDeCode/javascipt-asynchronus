Belajar Asynchronus

Asyn Await

Asyn dan Await adalah code yang mempermudah pembuatan promise

dengan Async Await kita bisa membuat function Async dengan gaya Sync

Async

Digunakan untuk menandakan bahwa function tersebut Asynchronus dan mengbalikan promise

Await

Digunakan untuk mengambil value dari promise hasil function await

Await hanya bisa digunakan dalam async function
