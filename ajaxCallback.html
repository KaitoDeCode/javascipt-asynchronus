Ajax Callback

Ajax biasanya digunakan untuk mengirim data atau menerima data ke sertver

Tiap request kita selalu ingin mendapatkan response dari server

kita tidak dapat langsung mendapatkan data dari server lewat ajax karena ajax berlangsung secara asynchronus, sehingga kita perlu menunggu proses ajax selesai dijalankan

untuk mendapatkan informasi dari ajax, kita perlu menggunakan AJAX Callback,yang akan dieksekusi setelah proses asynchronus ajax selesai dilakukan
