Belajar Asynchronus

Apa Itu Callback

Callback Adalah Proses memanggil kembali program yang ada didalam proses async

Callback Biasanya dibuat dalam bentuk function dan akan dijalankan ketika proses asynchronus selesai

Contoh Async Method

  1. setInterval
    digunakan untuk menjalankan program dalam proses async selama periode waktu tertenu
  2. setTimeout
    digunakan untuk menajalankan program sekali dalam waktu tertentu
